📘 O que é HTML?

HTML (HyperText Markup Language) é a linguagem usada para criar a estrutura de um site. É como o esqueleto de uma página web.

🧱 Estrutura básica

Todo site começa com uma estrutura base como esta:

<!DOCTYPE html> <html> <head> <title>Meu site</title> </head> <body> <h1>Olá Mundo</h1> </body> </html>

🏷️ Tags importantes

Algumas das principais tags que você vai usar:

  • <h1> → Título
  • <p> → Parágrafo
  • <a> → Link
  • <img> → Imagem

💻 Exemplo prático

Veja um exemplo simples:

<h1>Meu primeiro site</h1> <p>Estou aprendendo HTML!</p>

🎯 Exercício

Crie um arquivo HTML com:

  • Um título com seu nome
  • Um parágrafo dizendo o que você está aprendendo
