🎨 O que é CSS?

CSS (Cascading Style Sheets) é usado para estilizar o HTML. Com ele você muda cores, tamanhos, posições e deixa o site bonito.

🎯 Exemplo básico

p { color: blue; font-size: 20px; }

Este texto está estilizado com CSS!

📦 Trabalhando com caixas (div)

.box { width: 100px; height: 100px; background: blue; }

🎯 Exercício

Crie uma caixa com:

  • Cor vermelha
  • Largura: 150px
  • Altura: 150px
