⚡ O que é JavaScript?

JavaScript é a linguagem que adiciona interatividade ao site. Com ele você pode reagir a cliques, mudar textos e muito mais.

🎯 Exemplo básico

alert("Olá, mundo!");

Esse código mostra uma mensagem na tela.

🖱️ Interação com botão

function mostrarMensagem() { document.getElementById("texto").innerText = "Você clicou no botão!"; }

🎨 Mudando cores com JavaScript

function mudarCor() { document.body.style.backgroundColor = "lightblue"; }
